## Role

This package exposes a clock state provider and multiple replaceable clock views.

## Layers

- `ClockContextProvider` owns the current time and refresh cadence.
- `AnalogClock` and `DigitalClock` are standalone visualizations that accept ordinary wrapper element props.
- `InlineAnalogClock` is the inline SVG visualization for text-like usage.

## Example

```tsx
import {
  AnalogClock,
  ClockContext,
  ClockContextProvider,
  DigitalClock,
} from '@kitsuyui/react-clock'

function ClockExample() {
  return (
    <ClockContextProvider>
      <ClockContext.Consumer>
        {(date) => (
          <>
            <AnalogClock timezone="Asia/Tokyo" date={date} />
            <DigitalClock timezone="Asia/Tokyo" date={date} />
          </>
        )}
      </ClockContext.Consumer>
    </ClockContextProvider>
  )
}
```
