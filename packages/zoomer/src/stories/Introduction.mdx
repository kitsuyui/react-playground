## Role

This package scales content to fit the available layout area.

## Layers

- `SizedZoomer` is the pure fit-to-bounds primitive for known dimensions.
- `AutoZoomer` measures the parent and feeds that primitive.

## Example

```tsx
import { AutoZoomer } from '@kitsuyui/react-zoomer'

<AutoZoomer>Responsive content</AutoZoomer>
```
