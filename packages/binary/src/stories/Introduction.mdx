## Role

This package provides binary visualization primitives for inline content.

## Layers

- `Bits` is the structural primitive that maps a numeric value to bit elements.
- `SimpleBitString` is the plain text default visualization.
- `DetailedBitString` is the richer diagnostic visualization.

## Example

```tsx
import { DetailedBitString } from '@kitsuyui/react-binary'

<DetailedBitString value={0b10101010} length={8} />
```
