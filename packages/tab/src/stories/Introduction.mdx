## Role

This package provides the structure and state contract for tab interfaces without prescribing presentation.

## Layers

- `TabUIBase` is the core interaction primitive.
- `TabContext` exposes the selected tab state to content containers.
- `TabButtonContext` exposes per-button state to custom tab button implementations.

## Example

```tsx
import {
  TabButtonContext,
  TabContext,
  TabUIBase,
} from '@kitsuyui/react-tab'

// Provide your own TabBar, TabButton, and ContentBox components.
```
