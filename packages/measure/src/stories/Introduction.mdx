## Role

This package measures available space and visualizes it.

## Layers

- `MeasureBase` is the pure SVG visualization for known width and height.
- `AutoMeasure` measures the parent and renders that primitive.

## Example

```tsx
import { AutoMeasure } from '@kitsuyui/react-measure'

<AutoMeasure />
```
