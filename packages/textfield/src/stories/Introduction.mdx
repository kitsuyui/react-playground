## Role

This package provides semantic wrappers around text inputs and textareas with IME-aware input chunk handling.

## Layers

- `TextField` wraps the native text input shape and adds `onInputChunk`.
- `TextArea` wraps the native textarea shape and adds the same IME-aware behavior.
- Both components remain drop-in replacements for ordinary element props.

## Example

```tsx
import { TextArea, TextField } from '@kitsuyui/react-textfield'

function FormExample() {
  return (
    <>
      <TextField placeholder="Search" onInputChunk={(text) => console.log(text)} />
      <TextArea onInputChunk={(text) => console.log(text)} />
    </>
  )
}
```
