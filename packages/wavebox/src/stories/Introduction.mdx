## Role

This package provides animated size primitives for testing layout flexibility over time.

## Layers

- `WaveBoxProvider` owns the animated width and height state.
- `WaveBoxContext` exposes that state for custom compositions.
- `SizedBox` renders explicit dimensions, while `WaveBox` is the default composition built on the provider.

## Example

```tsx
import { WaveBox } from '@kitsuyui/react-wavebox'

<WaveBox minWidth={80} maxWidth={160} minHeight={40} maxHeight={120} />
```
