## Role

This package fits text into available layout bounds.

## Layers

- `SizedDekamoji` is the pure fit-text primitive for known bounds.
- `AutoDekamoji` measures the parent and feeds that primitive.

## Example

```tsx
import { AutoDekamoji } from '@kitsuyui/react-dekamoji'

<AutoDekamoji text="あいうえお" />
```
