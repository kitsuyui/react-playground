# @kitsuyui/react-biticon

Bit icon component for React.
This package is a React component that displays 128-bit data as an icon.

## Design role

This package is an identifier visualization primitive.
It is closer to an `img`/embedded graphic than to a styled widget.

## Demo

Storybook: https://react-playground.docs.kitsuyui.com/storybook/

## Installation

### npm

```sh
npm install @kitsuyui/react-biticon
```

### yarn

```sh
yarn add @kitsuyui/react-biticon
```

### pnpm

```sh
pnpm add @kitsuyui/react-biticon
```

## Usage

```tsx
import { BitIcon, UUIDIcon } from '@kitsuyui/react-biticon'
```

`BitIcon` and `UUIDIcon` accept ordinary `svg` props for their outer graphic element.
Use those props to control size, class names, inline style, and accessibility metadata.
