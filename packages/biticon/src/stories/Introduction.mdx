## Role

This package renders 128-bit identifiers as embedded SVG graphics.

## Layers

- `BitIcon` renders a bigint-backed identifier graphic.
- `UUIDIcon` adapts UUID strings to the same SVG primitive.

## Example

```tsx
import { UUIDIcon } from '@kitsuyui/react-biticon'

<UUIDIcon uuid="018fc860-4d10-7b70-b012-9ca7c3525a13" width="1em" height="1em" />
```
