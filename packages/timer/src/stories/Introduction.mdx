```tsx
import {
  TimerContextProvider,
  TimerContext,
  DefaultTimer,
  type TimerContextValue,
} from '../'

const Timer = () => {
  return (
    <TimerContextProvider>
      <TimerContext.Consumer>
        {(props: TimerContextValue) => <DefaultTimer {...props} />}
      </TimerContext.Consumer>
    </TimerContextProvider>
  )
}
```

## Installation

Each component is published as a separate package.

Use npm, yarn or pnpm to install.

```sh
$ npm install @kitsuyui/react-timer
```

```sh
$ yarn add @kitsuyui/react-timer
```

```sh
$ pnpm add @kitsuyui/react-timer
```

## Usage

This package provides two layers:

1. `TimerContextProvider` as the state/controller primitive
2. `DefaultTimer` as the default UI on top of that primitive

If you want a different UI, consume `TimerContext` and render your own component.
