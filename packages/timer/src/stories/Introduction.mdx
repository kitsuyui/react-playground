## Role

This package provides a timer state/controller primitive plus a replaceable default UI.

## Layers

- `TimerContextProvider` owns the countdown lifecycle and exposes it through `TimerContext`.
- `DefaultTimer` is the package-provided default UI on top of that primitive.

## Example

```tsx
import {
  DefaultTimer,
  TimerContext,
  TimerContextProvider,
  type TimerContextValue,
} from '@kitsuyui/react-timer'

function TimerExample() {
  return (
    <TimerContextProvider>
      <TimerContext.Consumer>
        {(context: TimerContextValue) => <DefaultTimer {...context} />}
      </TimerContext.Consumer>
    </TimerContextProvider>
  )
}
```
