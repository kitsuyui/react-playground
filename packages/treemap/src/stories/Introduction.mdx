## Role

This package lays out hierarchical values into treemap regions.

## Layers

- `SizedTreemap` is the layout primitive for known width and height.
- `AutoTreemap` measures the parent and feeds that primitive.
- `TreemapContext` exposes per-cell layout state for custom cell renderers.

## Example

```tsx
import { AutoTreemap } from '@kitsuyui/react-treemap'

<AutoTreemap nodes={[{ value: 3, label: 'A' }, { value: 2, label: 'B' }]} />
```
