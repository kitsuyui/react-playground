## Installation

Each component is published as a separate package.

Use npm, yarn or pnpm to install.

```sh
$ npm install @kitsuyui/react-treemap
```

```sh
$ yarn add @kitsuyui/react-treemap
```

```sh
$ pnpm add @kitsuyui/react-treemap
```
