## Installation

Each component is published as a separate package.

Use npm, yarn or pnpm to install.

```sh
$ npm install @kitsuyui/react-timer
```

```sh
$ yarn add @kitsuyui/react-timer
```

```sh
$ pnpm add @kitsuyui/react-timer
```
